---
description: 生成今日工作日报
allowed-tools: Bash(git*)
---

调阅今天的 git 日志和今天的提交，为今天的工作内容生成工作日报，要求：

1. 先通过 git config user.name 确认我是谁
2. **所有 git 命令必须加上 --author="$(git config user.name)" 参数，只查询当前用户的提交，严禁包含其他人的提交**
3. 生成2款，第1款是 10 段话的工作日报，第2款是将上面10段话合并（记住是合并不是缩减）为4段话
4. 表达出当前开发阶段中，我 **对项目进度的推进和贡献** ，要符合我作为主力开发选手（Owner）的身份
5. 用中文输出，格式清晰专业
